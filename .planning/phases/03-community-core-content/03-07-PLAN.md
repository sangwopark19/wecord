**useJoinCommunity.ts — Fix generateNickname to be fully defensive (line 10-18):**
Wrap the entire `supabase.functions.invoke` call in a try/catch so that network-level errors (connection refused, DNS failure) are caught and fall back to local generation. The existing `if (error || !data?.nickname)` check only handles clean supabase-js error returns, not thrown exceptions:
```tsx
async function generateNickname(): Promise<string> {
  try {
    const { data, error } = await supabase.functions.invoke('generate-nickname');
    if (error || !data?.nickname) {
      const randomNum = Math.floor(1000 + Math.random() * 9000);
      return `User#${randomNum}`;
    }
    return data.nickname as string;
  } catch {
    const randomNum = Math.floor(1000 + Math.random() * 9000);
    return `User#${randomNum}`;
  }
}
```

  ```tsx
  if (error.code === '23505') {
  ```
