**Hooks:**

1. **`useFanFeed(communityId: string, sort: 'latest' | 'popular', filter: 'all' | 'following' | 'hot')`**:
   - `useInfiniteQuery` with `queryKey: ['fanFeed', communityId, sort, filter]`
   - `initialPageParam: null as { createdAt: string; id: string } | null`
   - Query from `posts_with_nickname` (NEVER raw `posts`), `.eq('community_id', communityId).eq('author_role', 'fan')`
   - PAGE_SIZE = 15
   - Sort `latest`: `.order('created_at', { ascending: false }).order('id', { ascending: false })`
   - Sort `popular`: Use offset pagination capped at 3 pages (45 posts total). `.order('like_count', { ascending: false }).order('created_at', { ascending: false }).range(pageParam * PAGE_SIZE, (pageParam + 1) * PAGE_SIZE - 1)`. getNextPageParam returns undefined if page >= 2 or lastPage.length < PAGE_SIZE.
   - Cursor pagination for `latest`: if pageParam exists, `.or(\`created_at.lt.${pageParam.createdAt},and(created_at.eq.${pageParam.createdAt},id.lt.${pageParam.id})\`)`
   - Filter `all`: no additional filter
   - Filter `following`: `.in('author_cm_id', followingCmIds)` — fetch user's following list first via `supabase.from('community_follows').select('following_cm_id').eq('follower_cm_id', myCmId)`, then pass IDs. If no followings, return empty array.
   - Filter `hot`: `.gte('like_count', 10)` (hot threshold = 10 likes)
   - Also fetch whether current user has liked each post: after fetching posts, batch query `supabase.from('likes').select('target_id').eq('user_id', userId).eq('target_type', 'post').in('target_id', postIds)`. Merge `isLiked: boolean` into each post object.

2. **`useCreatePost()`**:
   - `useMutation`
   - Parameters: `{ communityId: string, content: string, mediaUris: string[], postType: 'text' | 'image' | 'video' }`
   - Steps: (a) If mediaUris.length > 0, compress each image via `expo-image-manipulator` (resize width 1080, compress 0.7, JPEG format), upload to `post-media` bucket at path `communities/${communityId}/${userId}/${Date.now()}_${index}.jpg` (or .mp4 for video). Collect public URLs. (b) INSERT into `posts` table: `{ community_id, author_id: userId, content, media_urls: uploadedUrls, post_type: postType, author_role: 'fan' }`. (c) Invalidate `['fanFeed', communityId]`.
   - Error handling: if upload fails, show toast with `post.compose.error.imageUpload` i18n key. If insert fails, show `post.compose.error.publish`.

3. **`useDeletePost()`**:
   - `useMutation`
   - Calls `supabase.from('posts').delete().eq('id', postId)`
   - Optimistic removal: `onMutate` removes post from all matching `['fanFeed', ...]` infinite query caches
   - `onError` restores snapshot
