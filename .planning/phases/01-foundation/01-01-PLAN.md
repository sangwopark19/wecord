    3. Create turbo.json:
       ```json
       {
         "$schema": "https://turbo.build/schema.json",
         "tasks": {
           "build": {
             "dependsOn": ["^build"],
             "outputs": [".next/**", "dist/**", ".expo/**"]
           },
           "dev": {
             "cache": false,
             "persistent": true
           },
           "lint": {
             "dependsOn": ["^lint"]
           },
           "typecheck": {
             "dependsOn": ["^typecheck"]
           },
           "test": {
             "dependsOn": ["^build"],
             "outputs": ["coverage/**"]
           },
           "clean": {
             "cache": false
           }
         }
       }
       ```

    4. Create .npmrc:
       ```
       auto-install-peers=true
       strict-peer-dependencies=false
       ```

    5. Update .gitignore to include:
       ```
       node_modules/
       .turbo/
       dist/
       .next/
       .expo/
       .env
       .env.local
       .env.*.local
       *.tsbuildinfo
       coverage/
       .DS_Store
       ```

    6. Run `pnpm install` to bootstrap turbo.
