## 1. 프로젝트 개요 & 비전

### 1.1 제품 비전

**Wecord**는 BL·GL 팬덤을 위한 올인원 팬 커뮤니티 플랫폼이다. 작가, 일러스트레이터, 성우 등 BL·GL 크리에이터와 팬이 직접 소통하고, 콘텐츠를 공유하며, 팬덤 활동을 기록하는 전용 공간을 제공한다.

> **한 줄 요약**: "BL·GL 크리에이터와 팬이 만나는 유일한 공간"

### 1.2 왜 Wecord인가

[Weverse 경쟁 분석 참조] Weverse는 K-pop 팬덤 올인원 플랫폼으로 178+ 아티스트, 12M+ MAU를 확보했으나, BL·GL 팬덤은 트위터, 디스코드, 픽시브 등에 분산되어 있다. 전용 플랫폼이 부재한 상황에서 Wecord는 다음의 기회를 포착한다:

- **분산된 팬덤 통합**: 트위터(소통) + 픽시브(작품) + 디스코드(커뮤니티) + 커머스가 분리된 현재 구조를 하나로 통합
- **글로벌 BL·GL 시장**: 한국, 일본, 태국, 중국이 4대 시장이며, 언어 장벽이 팬덤 확장의 핵심 장벽 [Weverse 분석: 해외 90% 유저, 자동번역이 리텐션 핵심 요인]
- **크리에이터 수익화 부재**: BL·GL 크리에이터는 패트리온, 판타지아 등에 의존하고 있으며 팬덤 통합 수익화 도구가 없음
- **서브컬처 확장성**: BL·GL에서 시작하여 보이스드라마, 웹소설, 웹툰, 코스프레 등 서브컬처 팬덤 전체로 확장 가능
<!-- 회의 반영 2026-03-12 -->
- **페르소나 분리**: 팬들은 아티스트마다 다른 닉네임/정체성을 원함. 커뮤니티별 독립 프로필로 차별화

## 2. 타겟 사용자

[Weverse personas.md 기반, BL·GL 팬덤에 맞게 조정]

### P1. 일반 팬 (Free Fan)

| 항목 | 내용 |
|------|------|
| 별칭 | "캐주얼 팬" |
| 연령대 | 15~35세 (Z세대~밀레니얼, 여성 비율 80%+) |
| 지역 | 글로벌 (한국, 일본, 태국, 중국, 영어권 집중) |
| 결제 성향 | 무료 기능 위주, 간헐적 굿즈 구매 |
| 핵심 니즈 | 좋아하는 BL·GL 작가/성우의 최신 소식 확인, 팬덤 소통, 번역된 콘텐츠 소비, **커뮤니티별 페르소나 분리** |
| 리텐션 요인 | 크리에이터 포스팅 빈도, 팬 커뮤니티 소속감, 자동번역 |

### P2. 프리미엄 팬 (Premium Fan)

| 항목 | 내용 |
|------|------|
| 별칭 | "코어 팬" |
| 연령대 | 18~40세 |
| 결제 성향 | 월 $9~30+ (멤버십 + DM + 디지털콘텐츠 + 굿즈) |
| 핵심 니즈 | 크리에이터와 1:1 소통, 한정판 콘텐츠, 멤버십 전용 혜택, 컬렉터블 수집 |
| 리텐션 요인 | DM 메시지, 멤버십 전용 콘텐츠, 매몰 비용(Jelly 잔액/배지) |

### P3. 크리에이터 (Creator)

| 항목 | 내용 |
|------|------|
| 별칭 | "공급자" — BL·GL 작가, 일러스트레이터, 성우, 번역가 등 |
| 활동 패턴 | 작품 연재/완결 주기에 따른 비정기적 포스팅, 팬 소통 |
| 핵심 니즈 | 팬 전용 소통 채널, 간편한 콘텐츠 공유, DM 수익, 프라이버시 보호 |
| 리텐션 요인 | 팬 반응 가시성, DM 수익 분배, 간편한 UX |

### P4. 운영/매니지먼트 (Admin)

| 항목 | 내용 |
|------|------|
| 별칭 | "관리자" — 에이전시, Wecord 운영팀 |
| 핵심 니즈 | 커뮤니티 관리, 콘텐츠 모더레이션, 공지 관리, 팬 데이터 인사이트 |
| 사용 인터페이스 | 웹 기반 관리자 대시보드 (Next.js) |

### 페르소나 간 관계

```
P1 일반팬 ──(전환)──→ P2 프리미엄팬
     │                      │
     └── 콘텐츠 소비 ←── P3 크리에이터
                             │
                        P4 운영 (관리/지원)
```

### 팬덤 퍼널 [Weverse 2025 Fandom Trend Report 참조]

```
Discovery    → 크리에이터 발견 (온보딩 큐레이션, 홈 추천)
Exploration  → 커뮤니티 가입 (커뮤니티별 닉네임 설정)
Appreciation → 댓글/좋아요, 번역, 팬↔팬 팔로잉
Interaction  → DM 구독, 멤버십, 커머스
Amplification→ 슈퍼팬, 팬아트, 팬덤 리더
```
