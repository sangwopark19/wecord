## 1. 글로벌 디자인 시스템

### 1.1 컬러 팔레트

| 용도 | 색상 코드 (추정) | 설명 |
|------|---------------|------|
| **배경 (Primary)** | `#000000` ~ `#0D0D0D` | 거의 순수 블랙, 전체 앱 배경 |
| **카드 배경** | `#1A1A1A` ~ `#1E1E1E` | 약간 밝은 다크 그레이, 카드/섹션 컨테이너 |
| **입력 필드 배경** | `#2A2A2A` ~ `#333333` | 텍스트 필드, 비활성 영역 |
| **텍스트 (Primary)** | `#FFFFFF` | 제목, 본문 텍스트 |
| **텍스트 (Secondary)** | `#999999` ~ `#AAAAAA` | 부제목, 날짜, 보조 텍스트 |
| **텍스트 (Tertiary)** | `#666666` ~ `#777777` | 비활성 탭, 안내 문구 |
| **액센트 (Teal/Cyan)** | `#00E5C3` ~ `#00D4AA` | CTA 버튼(충전), 토글 ON, 설정 값 표시 |
| **액센트 (Pink)** | `#FF6B9D` ~ `#FF69B4` | 아티스트별 테마 컬러(BLACKPINK), FAB 버튼 |
| **링크/하이퍼텍스트** | `#00E5C3` (Teal) | "더 보기", 링크 텍스트 |
| **경고/삭제** | `#FF4444` ~ `#FF3333` | 로그아웃, 서비스 탈퇴 텍스트 |
| **배지 (Notification)** | `#FF3B30` | 알림 뱃지 (빨간 점/숫자) |
| **구분선** | `#2A2A2A` ~ `#333333` | 카드 내부 구분선 (hairline) |

### 1.2 타이포그래피

| 용도 | 크기 (추정) | 두께 | 비고 |
|------|-----------|------|------|
| **앱 로고** | 커스텀 | Bold | "weverse" 소문자, 세리프 커스텀 폰트 |
| **페이지 타이틀** | 18–20sp | Bold | "더보기", "설정" 등 |
| **섹션 타이틀** | 16–18sp | Bold | "당신의 아티스트를 만나보세요" |
| **카드 타이틀** | 15–16sp | SemiBold | 아티스트명, 메뉴 항목 |
| **본문 텍스트** | 14–15sp | Regular | 포스트 본문, 설명 |
| **보조 텍스트** | 12–13sp | Regular | 날짜, 멤버 수, 부제 |
| **캡션/라벨** | 11–12sp | Medium | 탭바 라벨, 필터 칩, 배지 텍스트 |
| **작은 텍스트** | 10–11sp | Regular | 법적 고지, 사업자 정보 |

### 1.3 간격 시스템 (Spacing)

| 요소 | 값 (추정) |
|------|----------|
| **화면 좌우 패딩** | 16–20px |
| **카드 내부 패딩** | 16px (상하좌우) |
| **카드 간 간격** | 12–16px (세로) |
| **리스트 아이템 간격** | 0px (구분선으로 분리) |
| **섹션 간 간격** | 20–24px |
| **아이콘-텍스트 간격** | 8–12px |

### 1.4 모서리 반경 (Border Radius)

| 요소 | 반경 |
|------|------|
| **카드 컨테이너** | 12–16px |
| **배너/프로모션** | 12–16px |
| **프로필 아바타 (원형)** | 50% (완전 원형) |
| **버튼 (CTA)** | 24–28px (pill shape) |
| **칩/태그** | 16–20px (pill shape) |
| **커뮤니티 아이콘 (홈 상단)** | 12px (rounded square) |
| **입력 필드** | 8px |
| **모달/바텀시트** | 상단 16–20px |

### 1.5 아이콘 스타일

- **스타일**: Line icon (아웃라인), 일부 Filled
- **크기**: 탭바 24px, 헤더 액션 24px, 메뉴 아이콘 20–24px
- **색상**: 비활성 `#777777`, 활성 `#FFFFFF`
- **참고**: Lucide, Feather 스타일과 유사한 미니멀 라인 아이콘

## 2. 탭바 (Bottom Navigation)

### 레이아웃

```
┌──────────────────────────────────────────────┐
│   [W]홈     [🛒]Shop    [✈]DM     [•••]더보기  │
└──────────────────────────────────────────────┘
```

### 상세 스펙

| 속성 | 값 |
|------|-----|
| **탭 수** | 4개 |
| **높이** | ~56px (Safe Area 제외) |
| **배경** | `#000000` (완전 블랙) |
| **상단 경계** | 없음 (그림자나 보더 없이 배경색으로 분리) |
| **활성 탭 색상** | `#FFFFFF` (아이콘 + 텍스트) |
| **비활성 탭 색상** | `#777777` (아이콘 + 텍스트) |
| **활성 인디케이터** | 없음 (색상 변화만) |

### 각 탭 상세

| 순서 | 라벨 | 아이콘 설명 |
|------|------|-----------|
| 1 | **홈** | "W" 로고 마크 (위버스 심볼, 커스텀) |
| 2 | **Shop** | 쇼핑백 아이콘 (아웃라인) |
| 3 | **DM** | 종이비행기 아이콘 (아웃라인, Weverse DM 심볼) |
| 4 | **더보기** | 가로 점 3개 (•••) |

### 인터랙션
- 탭 전환 시 **크로스페이드** (애니메이션 최소)
- 현재 탭 재탭 시 최상단 스크롤

## 3. 홈 화면 — 커뮤니티 미가입 상태

> **스크린샷**: Image 1 (1메인화면_가입x_.PNG)

### 3.1 헤더

```
┌──────────────────────────────────────┐
│  [weverse 로고]              [🔔¹]   │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| **높이** | ~56px |
| **배경** | 투명 → 스크롤 시 `#000000` |
| **좌측** | "weverse" 로고 텍스트 (화이트, 센터 정렬) |
| **우측** | 알림 벨 아이콘 + 빨간 뱃지 숫자(1) |
| **뱃지** | 직경 ~16px, 빨간 원, 흰색 숫자, 아이콘 우상단에 걸침 |

### 3.2 커뮤니티 선택 바 (My Communities Bar)

```
┌──────────────────────────────────────┐
│  [+]  내가 좋아하는 아티스트를 찾아보세요!    │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| **컨테이너** | `#1A1A1A` 카드, radius 12px, 좌우 마진 16px |
| **높이** | ~64px |
| **+ 버튼** | 48×48px 원형, 그라디언트 배경 (보라→분홍→남색), 중앙에 + 아이콘(흰색) |
| **텍스트** | "내가 좋아하는 아티스트를 찾아보세요!" 흰색, 14–15sp |
| **정렬** | + 버튼 좌측, 텍스트 우측 (수직 중앙 정렬) |

### 3.3 프로모션 배너 캐러셀

```
┌──────────────────────────────────────┐
│                                      │
│     [배너 이미지/프로모션 카드]          │
│                                      │
│                            01 | 10   │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| **높이** | ~200px |
| **너비** | 화면 폭 – 좌측 마진 (우측은 다음 배너 살짝 노출) |
| **모서리** | radius 16px |
| **스와이프** | 가로 스크롤, 스냅 정렬 |
| **인디케이터** | 우하단 "01 \| 10" 텍스트 (현재/전체, 파이프 구분) |
| **인디케이터 스타일** | 반투명 배경 pill, 흰색 텍스트, 12sp |
| **자동 스크롤** | 있음 (약 4–5초 간격 추정) |
| **배너 디자인** | 풀 이미지 또는 그라디언트 + 텍스트 + 로고 조합 |

### 3.4 기능 프로모션 배너 (단일)

```
┌──────────────────────────────────────┐
│ LIVE · 리스닝 파티 채팅을 실시간 번역으로! │
│ 디지털 멤버십에서 바로 이용하세요    [🎧]  │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| **배경** | 그라디언트 (보라→분홍→남색, 좌→우 또는 대각선) |
| **높이** | ~80px |
| **모서리** | radius 12–16px |
| **좌측** | 텍스트 2줄 (타이틀 Bold 15sp + 설명 Regular 12sp) |
| **우측** | 일러스트 아이콘 (헤드폰 + 음표, 약 48×48px) |
| **마진** | 좌우 16px |

### 3.5 아티스트 추천 섹션

```
┌──────────────────────────────────────┐
│ 당신의 아티스트를 만나보세요     01 | 10  │
│                                      │

## 4. 홈 화면 — 커뮤니티 가입 상태

> **스크린샷**: Image 2 (2메인화면_가입o_.PNG)

### 4.1 헤더

미가입 상태와 동일 구조.

### 4.2 커뮤니티 선택 바 (가입 후)

```
┌──────────────────────────────────────────┐
│ [+] [BLACKPINK] [dearALICE] [ARIN] ...  │
└──────────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| **컨테이너** | `#1A1A1A` 카드, radius 12px |
| **높이** | ~88px (아이콘+라벨 포함) |
| **+ 버튼** | 미가입 상태와 동일 (48×48px 그라디언트 원형) |
| **커뮤니티 아이콘** | 48×48px, rounded square (radius ~12px) |
| **아이콘 배경** | 아티스트별 커스텀 이미지/로고 |
| **라벨** | 아이콘 하단, 12sp, 흰색, 중앙 정렬, 최대 1줄 + 말줄임 |
| **간격** | 아이콘 간 12–16px |
| **스크롤** | 가로 스크롤 (가입 커뮤니티 많을 시) |
| **정렬** | + 버튼이 항상 맨 좌측 고정 |

### 4.3 프로모션 배너 캐러셀

미가입 상태와 동일 구조이나 내용이 다름 (Weverse Shop 프로모션 등).

| 속성 | 값 |
|------|-----|
| **배너 예시** | "Weverse Shop" 라벨 + "위버스샵 현장수령 이용 가이드" + 일러스트 |
| **라벨 위치** | 카드 좌상단, "Weverse Shop" 텍스트 (13sp, Bold) |
| **타이틀** | 좌측 하단, Bold 18–20sp, 최대 2줄 |
| **서브 텍스트** | 타이틀 아래, Regular 12–13sp, 보조 색상 |
| **인디케이터** | "04 \| 10" 우하단 |

### 4.4 기능 프로모션 배너

```
┌──────────────────────────────────────┐
│ 실시간 채팅 번역, 멤버십 혜택으로 오픈!  │
│ 언어 걱정 없이 아티스트와 대화하세요 [Aa文]│

미가입 상태의 배너와 동일 레이아웃, 내용만 다름.

```
┌──────────────────────────────────────┐
│         더보기          [≡조절] [⚙•]  │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|

## 부록 C: 핵심 인터랙션 패턴 요약

### 스크롤 동작
- **커뮤니티 내부**: 히어로 이미지가 위로 축소되며 탭바가 상단에 sticky 고정
- **탭 내부**: 필터 바도 sticky 고정 (Fan 탭)
- **홈 화면**: 일반 수직 스크롤, 배너 영역은 가로 스와이프

### 터치 피드백
- **카드**: 터치 시 약간 어두워지는 ripple 효과
- **버튼**: 터치 시 opacity 0.7 또는 scale 0.97
- **탭**: 즉시 전환, 탭 인디케이터 슬라이드 애니메이션

### 로딩 패턴
- **스켈레톤 UI**: 카드/리스트 로딩 시 회색 박스 플레이스홀더 (추정)
- **무한 스크롤**: Fan/Artist 탭에서 하단 도달 시 자동 로드
- **Pull to Refresh**: 상단 당겨서 새로고침 (추정)

### 토스트/스낵바
- **위치**: 화면 하단, 탭바 바로 위
- **배경**: `#333333`, radius 8px
- **텍스트**: 14sp, 흰색
- **지속**: 2–3초 후 자동 소멸
- **예시**: "가입이 완료되었습니다."
